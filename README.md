# 📌 Project Overview

The **Bank Loan Report Analysis** project is an end-to-end Business Intelligence solution designed to help financial institutions monitor loan performance, evaluate customer risk, and make data-driven lending decisions.

Using **SQL Server** for data analysis and **Power BI** for interactive dashboards, this project transforms raw banking data into meaningful business insights that support executives, analysts, and decision-makers.

---

# 🎯 Business Problem

Banks process thousands of loan applications every month. Without proper analytics, it becomes difficult to:

- Monitor loan performance
- Identify risky customers
- Track repayment trends
- Evaluate lending efficiency
- Generate reports quickly
- Support strategic decision-making

This dashboard addresses these challenges through interactive visualizations and KPI tracking.

# 💡 Business Insights

- Majority of loans are successfully repaid.
- Good loans significantly outperform bad loans.
- Debt Consolidation is one of the most common loan purposes.
- Certain states receive higher loan funding than others.
- Customers with longer employment histories show better repayment behavior.
- Loan grade influences repayment performance.
- Monthly borrowing trends indicate seasonal demand.
- Interest rates vary across customer risk categories.

- Strengthen risk assessment for high-risk borrowers.
